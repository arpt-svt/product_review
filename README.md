# Product Review
API to add new product  
`/api/add/product/`  

API to list all product  
`/api/list/products/`  

API to delete a product      
`/api/delete/product/(?P<id>\d+)/`  

API to buy product  
`/api/buy/`  

API to view users purchase  
`/api/user/purchases/`  

API to give rating  
`/api/rate/product/`  

Data Format to send for rating
`{
	"product":1,
	"rate":4
}`
